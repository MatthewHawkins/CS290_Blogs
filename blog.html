Code Blog 1

This first assignment was a great introduction to getting used to HTML tags. Having played around with them a little bit beforehand, I had little trouble completing this assignment.

I learned quite a bit in lecture that I was able to implement in the assignment, namely:

  • h1 - h6 tags are not used simply for size, but to help divide up different heading sections
  • How to link both locally and non-locally hosted images
  • More familiarity with the paragraph tag instead of the div tag
While I knew a lot of the functionality of these tags going in, this was an excellent refresher.

I used a few different resources to help me in this assignment. Most of them were either notes provided by Dr. Hess, or articles online:

  • Dr. Hess's Google Doc with HTML notes
  • An article that discusses image tags in HTML
  • A git handbook to help remind me of the commands we needed to use

I'm looking forward to expanding on this knowledge in the next assignment, and re-learning how to implement CSS to style our pages!
