Challenges

This week's assignment ended up being relatively difficult for me for a few reasons. First, I found that some of the CSS that I was trying to apply didn't work in the way I expected. I found myself spending a lot of time looking up various problems that I was enountering on Google, and simply trial and error style solution hunting until I found something that worked. I think I'll need to try doing some CSS with a simpler HTML file, or one that I create in order to get a better grasp on the topic.

What I Learned

To be perfectly honest, in this assignment, I learned that I don't have a natural talent for CSS :) I did get a relatively decent grasp on how modify various class elements, and especially got used to margins, borders, and padding. The flexboxes challenged me, and justifying content within the boxes, as well as on the site as a whole, was the most difficult part.

Resources I Used

Things I Googled

The first thing I had to Google was how to properly use flexboxes. I was unsure of their use, and found an excellent resource that helped me figure out what keywords to use when I was attempting to fix issues. The second thing I spent a lot of time googling was how to size images correctly. I found a lot of advice saying that max-height and max-width were both good resources, but didn't know that I could enter a pixel value instead of a percentage value. For some reason, the percentage values were not working on every image correctly. I eventually was helped on Piazza with this issue.
