Challenges

I very much enjoyed this week's assignment. I think JavaScript is an awesome language (almost like Python and C++ had a lovechild). I found it relatively intuitive and forgiving, am looking forward to continuing to use it. The hardest part for me was figuring out how to apply the text filter - for some reason, I had a very hard time with this. Finally, deciding to create two arrays and then only remove an element if you hit the end of the array without finding the word seemed to be my best option. Still, other than that, the rest of it (after looking some functionality up) was fairly straightforward and enjoyable.

What I Learned

I learned everything I now know about JS in this assignment! I learned how important it is to be deliberate with your HTML code (I can see how poot HTML would make JS a royal pain) and how to manipulate the DOM using various JS functionality. Additionally, I learned how to make a page dynamic and responsive, and not just a glorified slideshow. I can now also see the importance of learning how to manipulate data so that you can compare it to values already present (like we had to do with the filters).
